# Module 4 Project: API Explorer & Design Document

## Setup

1. Activate your virtual environment:
   ```bash
   source venv/bin/activate   # Mac/Linux
   venv\Scripts\activate      # Windows
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## What You're Building

A two-part project:

**Part 1 — API Exploration** (`part1_api_exploration.py`)
Consume 3 public APIs and document what you discover. For each API: make multiple requests, explore the response structure, handle errors.

**Part 2 — API Design** (`part2_design_template.md`)
Design a REST API for a "Pet Adoption Service" app from scratch.

## The 3 APIs to Explore (Part 1)

| API | Base URL | Auth Required |
|-----|----------|---------------|
| JSONPlaceholder | https://jsonplaceholder.typicode.com | No |
| PokeAPI | https://pokeapi.co/api/v2/ | No |
| REST Countries | https://restcountries.com/v3.1/ | No |
